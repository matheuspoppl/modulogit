- git log - resulta no historico de alterações do repoditorio - git status - resulta na situação atual de alterações dos arquivos do repoditório - git init - para iniciar o git - git commit -am "comentário" - adiciona e atualiza as alterações feitas no arquivos - git reset --hard (hash do commit) - volta o repositório para a situação do commit(hash) usada deletando tudo o que aconteceu no repositório depois daquele momento, volta pra situação daquele momento (hash usada) - git reset --soft (hash do commit) - volta o repositório para a situação do commit(hash) usada mantendo as alterações dos arquivos, porém sem comitá-las, volta pra situação daquele momento (hash usada) antes de ter sido salvo - git branch - resulta na lista de todos os branchs que existem e o que tiver com * é o que está sendo usado no momento, é o atual que eu estou - git branch (nome do branch) - cria um novo branch com o nome citado importando todos os arquivos e alterações feitas do branch a partir do qual o novo branch foi gerado, ou seja, cria uma cópia do branch atual com o nome citado. - git checkout (nome do branch) - troca de branch para o branch citado
